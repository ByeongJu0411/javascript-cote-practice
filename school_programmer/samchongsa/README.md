삼총사 문제 풀이

이번 문제는 정수 번호를 가진 학생들 중 서로 다른 3명을 골라 합이 0이 되는 경우의 수를 반환합니다.
이전 과제에서 풀었던 블랙잭과 동일하게 3중for문을 사용하여 조합을 완전 탐색하는 방식으로 해결했습니다.

i < j < k 조건을 만족하도록 반복문을 구성하여 서로 다른 3명만 선택되도록 했습니다.

for (let i = 0; i < n; i++) {
for (let j = i + 1; j < n; j++) {
for (let k = j + 1; k < n; k++) {
if (number[i] + number[j] + number[k] === 0) answer++;
}
}
}

문제를 풀어보니, 블랙잭과 구조가 거의 동일함을 느꼈습니다.
조건만 다르지 3중 for문으로 조건을 체크하는 문제입니다.

여기서 "서로 다른 학생의 번호가 같을 수 있다"는 것은 값이 아니라 인덱스 기준으로 중복을 방지해야함을 이해했습니다.
